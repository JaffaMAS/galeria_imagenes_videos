# Galería de Imágenes y Videos

Aplicación móvil en Flutter para Android que permite explorar imágenes y videos del dispositivo.

## Objetivo
Crear una galería simple, estable y optimizada para Android, con soporte para la mayoría de formatos comunes.  
Versión inicial: MVP con acceso a medios, grid de thumbnails y visualizador básico.

## Dependencias principales
- photo_manager: ^3.8.3
- video_player: ^2.10.1
- permission_handler: ^12.0.1
- extended_image: ^3.0.2
- path_provider: ^2.1.5
